# Moodiary v2.7.3 - 修复版

## 📋 发布说明

Moodiary 是一款很好用的日记软件，原作者很长时间没有更新了。我使用中发现了一些 bug，在社区也发现了一些 bug 和新需求，但无人维护。我在原作者的基础上，修复了以下内容：

### ✨ 新增功能
- ✅ **添加自定义日记封面功能** - 可以为每篇日记设置独特的封面图片
- ✅ **添加自定义日记背景功能** - 支持自定义日记背景，让日记更加个性化

### 🐛 问题修复
- ✅ **修复使用 WebDav 同步时图片不加密的问题** - 确保通过 WebDav 同步的图片数据得到加密保护

### 📦 下载信息

- **文件名**: `moodirry-release.apk`
- **版本**: 2.7.3 (build 90)
- **大小**: 61 MB
- **架构**: arm64-v8a (64位 ARM)
- **最低 Android 版本**: Android 7.0 (API 26)

### 📱 安装方法

1. 下载 `moodirry-release.apk` 文件
2. 在 Android 设备上允许安装来自未知来源的应用
3. 打开 APK 文件并安装

### ⚠️ 注意事项

- 本版本为修改版本，非原作者官方发布
- 建议在安装前备份原有数据
- 如遇到问题，欢迎提 Issue 反馈
